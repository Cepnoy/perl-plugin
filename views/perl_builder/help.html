This is smart builder for perl based applications

  • goes through svn/ directories - aka 'projects'
  • founds last tag if possible for every project
    • for every project/tag : makes installation and optionally creates distributive, namely:
    • installs from source code with `cpanm . ` command into $WORKSPACE/cpanlib
      • if project == 'app/'
      • makes distributive
      • mixes in $WORKSPACE/cpanlib into distributive
      • copies all to artifacts directory
